# Satisfactory Modeler Database for Periodic Madness

This repo contains the custom [Satisfactory Modeler](https://satisfactorymodeler.itch.io/satisfactorymodeler)
database for the [Periodic Madness](https://ficsit.app/mod/PeriodicMadness) Satisfactory mod.

## Usage

- Install a separate installation of Satisfactory Modeler to not mess up your existing one.
- Clean out the `images/icons/` folder of your new Satisfactory Modeler installation.
- Go to the [releases](https://github.com/QuingKhaos/satisfactory-modeler-pm/releases) page.
- Download the database archive matching your installed mod version.
- Extract the database archive into your new Satisfactory Modeler installation, overwriting existing files.
